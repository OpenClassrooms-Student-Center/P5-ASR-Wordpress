# Worpress 5.7

## Procédure d’installation

Pour installer notre application web, voici les prérequis :    
- Ubuntu 20.04
- Apache
- PHP
- MySQL ou MariaDB

Pour installer les différentes briques logicielles, vous pouvez suivre la documentation créer par Ubuntu : 
- Installation des prérequis : https://doc.ubuntu-fr.org/lamp#installation 
- Installation de Wordpress : https://doc.ubuntu-fr.org/wordpress

Vous aurez à la suite de cela, un Wordpress totalement fonctionnel, nous nous occuperons du reste avec nos accès FTP.
